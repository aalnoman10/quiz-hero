  1. What is the difference between map(), filter(), forEach() and find() ?

    map() : Executes a given function once for each element of an array.It calls a function for each array element and Creates a new array with return values.

    filter() : Executes a given function once for each element of an array. The function has a condition and returns an array containing each element that satisfies the condition.

    forEach() : Executes the given function once for each element of an array Each method returns nothing.

    find() : Executes a given function once for each element of an array. The function has a condition and returns the first element containing the condition.

  2. What is the difference between localStorage sessionStorage and Cookies ?

    localStorage : Local storage where user data is stored.Even if the page is closed, it is not deleted. Because the data is saved in the browser.

    sessionStorage : Session storage is where user data is stored for a period of time, which is automatically deleted when the page is closed

    Cookies : Cookies are a type of data, which are stored in the browser in small text files.
